Information regarding Authentication Modes while availing any service

  • Authentication modes for availing any service on Vahan Portal is as per directions from respective state and accordingly authentication modes are enabled or disabled.
  • So while availing any service wherever Aadhaar biometric authentication is only vailable mode then applicant may visit the nearest CSC or attach a biometric device to the laptop/ desktop to get authenticated.
  • If you do Aadhaar Authentication with Aadhaar number of a person other than registered owner then your will be rejected and fees shall not be refundable.
  • For Services not in the below list, user has to visit RTO for verification with appointment(if required).
  • For further details, Click on User Manual icon on top right of Home Page.
